# Pattern2Scala

This is the [PatternRepeater](https://github.com/chicarrida/PatternRepeater) in Scala<br>
Click to place a shape

Keys:<br>
1 -> Rectangle <br>
2 -> Hexagon <br>
3 -> Triangle <br>
4 -> Thingy(supposed to be an Ellipse like thing...)<br>


r/R -> Rotate<br>
c/C -> Clear<br>
i/I -> show initial shapes<br>
